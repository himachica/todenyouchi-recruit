2002年入社

I.M

経営企画部 労務人事G

Career Story.01

ライフイベントもキャリアも、あきらめない。
制度と仲間に支えられ、キャリアを重ねてきたリアルストーリー。

  1. 1年目

    人事の世界へ。
    社会人としての第一歩

    新卒で配属されたのは人事部。給与や制度運用など、会社の基盤を支える仕事に取り組みました。分からないことばかりでしたが、先輩たちのサポートを受けながら、一つひとつできることを増やしていく毎日。人事という立場を通じて「社員が安心して働ける環境をつくる」というやりがいを知ったのが、この頃の大きな収穫でした。

  2. 3年目

    人事からの視野拡大。
    現場での挑戦

    長く人事を経験した後、東日本大震災を機に東京電力へ出向。賠償業務など社会的に大きな責任を担う仕事を経験しました。その後、千葉支社へ配属され、配電用地の業務に挑戦。バックオフィスからフロントの現場まで、多様な業務に関わることで、会社全体の仕事の流れを肌で感じられるようになりました。

  3. 10年目

    コールセンター立ち上げと、
    人生の転機

    千葉での勤務後、新設のコールセンターの立ち上げに参画。多くのパートスタッフとともにゼロから仕組みを作り上げていく中で、チームで動く楽しさを実感しました。ちょうどその時期に妊娠が分かり、産休・育休を取得。育児と仕事の両立という新しい課題に直面する一方で、周囲の理解や支えに助けられ、安心して出産に臨むことができました。

  4. 12年目

    両立の難しさと、
    支え合い

    育休から復帰後は、再びコールセンターで勤務。子どもの体調不良や保育園の事情など、不測の事態も多くありましたが、時短勤務制度や職場の協力で乗り越えることができました。「効率的に仕事を進める力」や「人に頼る勇気」を身につけたのは、この時期の大きな成長です。家庭と仕事、どちらも大切にしながらキャリアを築いていく意識が強まりました。

  5. 23年目

    人を支え、
    制度を整える立場へ

    現在は経営企画部で、労務人事グループと総務グループを兼務。給与計算や人事制度の運用・改定に携わりながら、社員一人ひとりが安心して働ける環境づくりに尽力しています。自身の経験を活かし、制度をさらに充実させることが使命だと感じています。「両立をあきらめず、自分らしい働き方を選べる会社でありたい」
    ――その思いが、今のキャリアの原動力です。

Message 学生の皆さんへ

前向きに学ぶ人に、
必ずチャンスは訪れる

ライフイベントと仕事の両立に不安を感じる方も多いと思います。私自身もそうでした。でも、この会社には理解してくれる仲間と制度があります。大切なのは、自分の気持ちを勇気をもって伝えること。前向きに学び続ける姿勢があれば、必ず成長できる環境です。一緒に働きやすい職場をつくっていきましょう。
